## Run Locally

Clone the crypto tracker

```bash
  git clone https://github.com/saurabh-619/portfolio-2021.git
```

Go to the client-v1 directory

```bash
  cd client-v1
```

Install dependencies

```bash
  yarn install
```

Start the gatsby site

```bash
  yarn clean
  yarn develop
```
